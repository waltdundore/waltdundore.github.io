Quick Start (15 Minutes)

1

Install Prerequisites

# On Mac (using Homebrew)
brew install git vagrant virtualbox

# On Linux (Ubuntu/Debian)
sudo apt install git vagrant virtualbox

# On Linux (Fedora)
sudo dnf install git vagrant VirtualBox

Mac users: Install Homebrew first

2

Clone Repository

git clone https://github.com/waltdundore/ahab.git
cd ahab
3

Choose Your OS & Deploy

# Option 1: Use default (Fedora 43)
make install

# Option 2: Choose your OS
echo "DEFAULT_OS=debian" > ahab.conf  # or ubuntu
make install

# Deploy Apache web server
make install apache

# Access at http://localhost:8080

Supports Fedora 43 (default), Debian 13, and Ubuntu 24.04 LTS

Frequently Asked Questions

Is this production-ready?

No. This is alpha software in active development. We're testing thoroughly and building toward production use for schools while documenting the process.

Yes. Free for K-12 schools, non-profits, and educational institutions under CC BY-NC-SA 4.0 license. For-profit entities need to negotiate commercial terms.
