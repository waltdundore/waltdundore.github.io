Quick Start (15 Minutes)

1

Install Prerequisites

# On Mac (using Homebrew)
brew install git vagrant virtualbox

# On Linux (Ubuntu/Debian)
sudo apt install git vagrant virtualbox

# On Linux (Fedora)
sudo dnf install git vagrant VirtualBox

Mac users: Install Homebrew first

2

Clone Repository

git clone https://github.com/waltdundore/ahab.git
cd ahab
3

Choose Your OS & Deploy

# Option 1: Use default (Fedora 43)
make install

# Option 2: Choose your OS
echo "DEFAULT_OS=debian" > ahab.conf  # or ubuntu
make install

# Deploy Apache web server
make install apache

# Access at http://localhost:8080

Supports Fedora 43 (default), Debian 13, and Ubuntu 24.04 LTS

Frequently Asked Questions

Is this safe for schools to use?

NO. This is experimental alpha software.

For school superintendents and IT directors: This software is NOT ready for your production network. We are actively developing and testing it. When we think it might be safe for schools, we will clearly announce that. But even then - don't trust us. Test it yourself in an isolated environment first.

We do not manage your network. You are responsible for evaluating any software before deploying it in your school environment.

Current status: Alpha development. Use only for learning and experimentation on isolated test systems.

Is it really free for schools?

Yes. Free for K-12 schools, non-profits, and educational institutions under CC BY-NC-SA 4.0 license. For-profit entities need to negotiate commercial terms.

What if I find a bug?

Report it on GitHub Issues. We document all issues transparently.

Can I contribute?

Yes! Fork the repository, make changes, submit pull requests. See Development Rules.
