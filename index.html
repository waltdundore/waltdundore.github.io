Quick Start (15 Minutes)

1

Install Prerequisites

# On Mac (using Homebrew)
brew install git vagrant virtualbox

# On Linux (Ubuntu/Debian)
sudo apt install git vagrant virtualbox

# On Linux (Fedora)
sudo dnf install git vagrant VirtualBox

Mac users: Install Homebrew first

2

Clone Repository

git clone https://github.com/waltdundore/ahab.git
cd ahab
3

Choose Your OS & Deploy

# Option 1: Use default (Fedora 43)
make install

# Option 2: Choose your OS
echo "DEFAULT_OS=debian" > ahab.conf  # or ubuntu
make install

# Deploy Apache web server
make install apache

# Access at http://localhost:8080

Supports Fedora 43 (default), Debian 13, and Ubuntu 24.04 LTS

Learning Questions

Do I need prior experience?

Nope. If you can use a computer and aren't afraid of typing commands, you can do this.

Is it really free?

Yes. No subscriptions, no "premium" version, no credit card required. Just download and use it.

Can I break my computer?

No. Everything happens inside a virtual machine that you can delete and recreate anytime.

What will I build?

Web servers, databases, and other services that you'd actually find running on real servers.

How long does it take?

About 30 minutes to get your first web server running. Getting comfortable with everything takes longer, obviously.
