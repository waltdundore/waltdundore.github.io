Two ways to try Ahab: Web interface (recommended for beginners) or command line

Web Interface

Point-and-click. Perfect for students and educators.

# Clone both repos
git clone https://github.com/waltdundore/ahab.git
git clone https://github.com/waltdundore/ahab-gui.git

# Start the GUI
cd ahab
make ui

# Open http://localhost:5001

See screenshots →

Command Line

Fast and scriptable. For automation and advanced users.

# Clone repo
git clone https://github.com/waltdundore/ahab.git

# Deploy Apache
cd ahab
make install apache

# Visit http://localhost:8080

Complete tutorial →

Prerequisites: macOS or Linux with Docker installed. Takes 5-15 minutes.
