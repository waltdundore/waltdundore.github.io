Before You Start: Install Prerequisites

You need Git, Vagrant, and VirtualBox installed first.

Install Prerequisites →
git clone https://github.com/waltdundore/ahab.git && cd ahab → Get the code
make install → Create your development environment
make install apache → Deploy your first web server
make test → Validate everything works

System requirements?

Minimum: 8GB RAM, macOS/Linux/Windows+WSL2

Required Software: Git, Vagrant, VirtualBox (all free)

Time: 15 minutes to get started, hours to master

⚠️ IMPORTANT: You must install Git, Vagrant, and VirtualBox BEFORE running make install. See installation guide →
